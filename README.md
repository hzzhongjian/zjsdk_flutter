# zjsdk_flutter

## 支持的广告类型

* 开屏广告
* 激励视频
* 插屏广告
* 横幅广告
* 信息流广告
* 视频流广告
* 视频内容
* 短剧
* H5页面


## 开始接入

> **插件需要`Flutter >=3.0.0`，请升级Flutter版本**
> **`enhanced Enum`需要指定`SDK>=2.17.0`，请调整SDK版本**
> **SDK版本调整后依然报错，请手动删除`.dart_tool`目录重新执行`pub-get`**
>

```yaml
environment:
  sdk: ">=2.17.0 <3.0.0"
```
